WordWise · Diccionario Inteligente
Diccionario web en inglés con definiciones, pronunciación, sinónimos y ejemplos de uso. Construido con HTML, CSS y JavaScript puro, consumiendo la API gratuita dictionaryapi.dev.

Características
- Búsqueda instantánea con autocompletado y debounce
- Tres vistas: Definiciones, Sinónimos y Ejemplos
- Pronunciación en audio (cuando está disponible)
- Tema claro/oscuro automático o manual
- Selector de tipografías (Inter, Georgia, Courier, Segoe UI, Trebuchet)
- Persistencia de última palabra, tema y fuente en localStorage
- Diseño responsive para todos los dispositivos

Instalación
1. Clona o descarga el repositorio
2. Abre index.html en tu navegador
No requiere dependencias ni servidor.

Tecnologías
- HTML5 semántico
- CSS3 con variables y modo oscuro
- JavaScript Vanilla (ES5+)
- Free Dictionary API

Uso
1. Escribe una palabra en inglés en el buscador
2. Presiona Buscar o Enter
3. Navega entre Definiciones, Sinónimos y Ejemplos
4. Haz clic en 🔊 para escuchar la pronunciación
5. Personaliza el tema y la fuente en el menú de configuración

Personalización
El tema y la fuente se guardan automáticamente en tu navegador
El modo Auto sigue la preferencia del sistema operativo
